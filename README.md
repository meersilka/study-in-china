# Сайт «Образование в Китае» — инструкция

Информационный сайт агентства по обучению в Китае: вузы, стипендии, отзывы, о компании
и слоты под рекламу (Яндекс РСЯ / Google AdSense) + аналитику (Яндекс.Метрика / GA4).

Чистый **HTML/CSS/JS без сборки** — открывается двойным кликом, деплоится куда угодно.

## 📁 Структура

```
china_edu_site/
├── index.html            ← главная страница (весь контент здесь)
├── privacy.html          ← политика конфиденциальности (для рекламы и 152-ФЗ)
├── assets/
│   ├── css/styles.css     ← все стили (палитра, тёмная тема, адаптив)
│   └── js/main.js         ← интерактив (меню, фильтр, формы, cookie)
├── CONTENT-CHECKLIST.md   ← список материалов для наполнения
└── README.md              ← этот файл
```

## ▶️ Как открыть

Двойной клик по `index.html`. Для корректной работы шрифтов нужен интернет
(шрифты Google Fonts; без интернета подставится системный шрифт — тоже нормально).

## ✍️ Как заменить контент

- **Тексты** — прямо в `index.html`, ищите нужный блок по заголовку (в файле есть комментарии-подсказки `ЗАМЕНИТЬ:`).
- **Фото** — положите файл в `assets/img/` и замените блок
  `<div class="media-ph">…</div>` на `<img src="assets/img/имя.jpg" alt="описание" loading="lazy" />`.
- **Видео** — в модалке `#videoModal` (внизу `index.html`) замените плейсхолдер на `<iframe>`
  с YouTube / VK / Rutube (пример есть в комментарии там же).
- **Название/контакты/соцсети** — «Company», телефоны, e-mail и ссылки в шапке и футере.

## 📊 Реклама и аналитика

Все коды **закомментированы** в `<head>` файла `index.html` и в `main.js` (функция `loadAnalytics`).
Когда появятся ID — вставьте их и раскомментируйте:

| Инструмент       | Что вставить                         | Где            |
|------------------|--------------------------------------|----------------|
| Яндекс.Метрика   | номер счётчика вместо `XXXXXXXX`      | `<head>` + `loadAnalytics()` |
| Google Analytics | `G-XXXXXXX` (Measurement ID)         | `<head>`       |
| Яндекс РСЯ       | код блока `R-A-XXXXXXX-1` из кабинета | блоки `.ad-slot` |
| Google AdSense   | `ca-pub-…` и `data-ad-slot`          | блоки `.ad-slot` |

> Аналитика грузится **только после согласия** в cookie-баннере — так требует 152-ФЗ.
> Для рекламы через РСЯ/Директ маркировку «Реклама» (erid) Яндекс ставит сам.

## 📨 Приём заявок

Сейчас формы показывают «спасибо», но данные **никуда не уходят** (статический сайт).
Чтобы получать заявки, выберите один способ и допишите отправку в `main.js` (блок 11):

1. **Formspree / Getform** — самый простой: регистрируете форму, получаете URL,
   ставите его в `action` формы. Без кода.
2. **Telegram-бот** — `fetch` на `api.telegram.org/bot<TOKEN>/sendMessage`.
3. **Битрикс24 / CRM** — отправка на входящий вебхук.

## 🚀 Как опубликовать (бесплатно)

- **Netlify** или **Vercel** — перетащить папку в браузер, готово (для РФ-аудитории может тормозить).
- **Timeweb / Reg.ru / Beget** — российский хостинг, оптимально для Яндекс-рекламы и по 152-ФЗ.
- **GitHub Pages** — бесплатно, но домен зарубежный.

Для рекламы в Яндексе и хранения данных россиян в РФ — берите **российский хостинг + домен .ru**.

## 🎨 Дизайн

Палитра «Фарфор и киноварь» + тёмная тема (переключатель в шапке, запоминается).
Меняется через CSS-переменные в начале `styles.css` (`:root` — светлая, `[data-theme="dark"]` — тёмная).
