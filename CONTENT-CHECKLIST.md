# Чек-лист материалов для сайта «Обучение в Китае»

> Заполните и пришлите файлы — мы разместим их на сайте в правильных местах.
> Форматы: тексты — в этом файле или в Word/Google Docs; фото — JPG/PNG; видео — ссылка на YouTube/VK/Rutube **или** файл MP4.

## 1. О компании
- [ ] Название компании и слоган (1 строка)
- [ ] Короткое описание «о нас» (3–5 предложений)
- [ ] Год основания, сколько студентов уже отправили
- [ ] Логотип (PNG на прозрачном фоне, если есть)
- [ ] Фото офиса / команды (2–4 шт.)
- [ ] Контакты: телефон, e-mail, мессенджеры (WhatsApp/Telegram/WeChat), адрес, соцсети

## 2. Университеты (по каждому вузу)
- [ ] Название (рус + англ)
- [ ] Город
- [ ] 3–5 сильных направлений/специальностей
- [ ] Стоимость обучения в год (примерно)
- [ ] Какие стипендии доступны
- [ ] 2–3 фото кампуса (горизонтальные, от 1200 px по ширине)
- [ ] Видео о вузе (ссылка или файл) — необязательно, но желательно

## 3. Отзывы клиентов
По каждому отзыву:
- [ ] Имя студента (можно только имя)
- [ ] Университет / город
- [ ] Текст отзыва (2–4 предложения)
- [ ] Фото студента (по желанию)
- [ ] Видео-отзыв (ссылка или файл) — если есть, это сильно повышает доверие

## 4. Образование в Китае (если есть свои материалы)
- [ ] Ваши тексты про стипендии / поступление / визы (если хотите заменить наши)
- [ ] Цифры и факты, которыми хотите гордиться (кол-во партнёрских вузов и т.п.)

## 5. Реклама и аналитика (когда будете готовы)
- [ ] Номер счётчика **Яндекс.Метрика** (для отслеживания рекламы в Яндекс.Директ)
- [ ] Идентификатор **Google Analytics** (G-XXXXXXX), если нужен
- [ ] Данные кабинета **Яндекс РСЯ** / **Google AdSense** — если хотите показывать рекламу и зарабатывать
- [ ] Реквизиты для «Политики конфиденциальности» (юр. лицо, ИНН)
